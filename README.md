# solvedac-cards

Generates solved.ac cards

## Usage

Base URL: `https://solvedac-cards-starcea.paring.moe/`

- `/profile/:handle`

  - Show solved.ac profile

  - `?size={int}`
    - Set card's height (default: 100px, valid range: 100px ~ 500px)
